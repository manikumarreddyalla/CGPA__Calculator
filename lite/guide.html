Calculator Guide

How SGPA and CGPA are estimated in this tool.

Calculator FAQ About Contact Privacy Terms

This CGPA calculator helps Vel Tech students estimate semester performance from subject credits and grades. It uses the common grade point method: each subject grade point is multiplied by the subject credit, all weighted points are added, and the total is divided by total credits.

SGPA Formula

SGPA = total of (credit x grade point) / total credits for one semester.

Example: if a semester has 20 credits and your weighted grade points total 180, the SGPA is 180 / 20 = 9.00.

CGPA Formula

CGPA = total weighted grade points across saved semesters / total saved credits.

Save each completed semester separately so the final CGPA includes all selected subjects and credits.

Grade Scale Used

The tool currently uses S/O = 10, A = 9, B = 8, C = 7, D = 6, and RA = 0. If your official regulation or department uses a different scale, compare the result with your university records before relying on it.

Using the Tool

  1. Search by subject code, shortcut, or subject name.
  2. Select your department filter if you want to narrow the subject list.
  3. Add subjects to the current semester and select the grade for each subject.
  4. Review the SGPA and total credits, then save the semester.
  5. Repeat for other semesters to calculate CGPA.
